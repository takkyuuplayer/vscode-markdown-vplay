# markdown-vplay

Runs V code in markdown and output the result.

## Features

![Runs V code](images/feature-runs-v-code.gif)

## Requirements

None.

## Extension Settings

* `markdown-vplay.workdir`: The working directory to run V script. The default is the same as the markdown file.
